# Magisk Loli Font Compat

1. Download [releases](https://github.com/muink/magisk-loli-font/releases)
2. Add to Magisk Module
3. Reboot your phone
4. Enjoy~

## FAQ

1. rescue bootloop
   - [KernelSU](https://kernelsu.org/guide/rescue-from-bootloop.html#brick-by-modules)
   - [Apatch](https://apatch.dev/rescue-bootloop.html)
   - Boot into custom recovery, run `touch /data/adb/modules/magisk-loli-font/disable` to disable module
